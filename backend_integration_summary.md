## 🚀 如何使用

### 1. 启动应用
```bash
python app_sf_style_final.py
```

### 2. 访问前台
打开浏览器访问: http://localhost:8006/
- 输入快递单号进行查询
- 支持多家快递公司

### 3. 访问后台
1. 点击导航栏的"管理后台"按钮
2. 使用账号 `admin` / `admin123` 登录
3. 在仪表板中管理理货单数据

### 4. 主要功能操作

#### 文件上传
1. 进入"文件上传"页面
2. 选择CSV或Excel文件
3. 可选择"仅预览"模式查看数据
4. 点击上传处理文件

#### 理货单管理
1. 进入"理货单管理"页面
2. 使用搜索框查找特定记录
3. 点击编辑按钮修改数据
4. 选择多条记录进行批量删除
